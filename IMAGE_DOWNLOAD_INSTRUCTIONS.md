## How to Download

1. Visit each Unsplash URL above
2. Click "Download" button on the photo page
3. Save the image as `cover.jpg` in the corresponding directory
4. Alternatively, you can right-click the image and "Save image as..."

## Alternative: Use Unsplash API

If you have Unsplash API access, you can download images programmatically using the photo ID from the URL.
