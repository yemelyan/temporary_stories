# Image Download Guide

## Quick Instructions

Each story needs a cover image. The Unsplash URLs and photographer credits are provided below. 

**Option 1: Use Keystatic Admin Panel (Recommended)**
1. Go to `/keystatic` in your browser
2. Edit each story
3. Upload images directly - Keystatic will handle the paths automatically

**Option 2: Manual Download**
1. Visit each Unsplash URL below
2. Download the image
3. Save as `cover.jpg` in the corresponding folder
